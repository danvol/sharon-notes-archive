# Visual Studio Code

## 常用配置

```json
// 显示空格
"renderWhitespace": "all"
// 恢复窗口
"Restore Windows": "none"
// 不打开上次窗口
"window.restoreWindows": "none"
// 显示空白字符
"editor.renderWhitespace": "all"
// 控制台允许输入
"code-runner.runInTerminal": true
```
