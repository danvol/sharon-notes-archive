- c：串行设备（一次性读取），如键鼠。

### 5.2.2 如何改变文件属性与权限

**`chgrp` - 修改所属组：**

```shell
chgrp [-R] dirname/filename ...
```

**`chown` - 修改所属用户：**

```shell
chown [-R] 账号名称 文件或目录
chown [-R] 账号名称:组名 文件或目录
chown user.group file
```

**`chmod` - 修改权限：**

```shell
chmod [-R] 644 文件或目录
chmod [ugoa] [+-=] [rwx],多个逗号隔开
```
